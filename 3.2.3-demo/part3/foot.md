# 页脚

<https://plugins.gitbook.com/plugin/page-footer-ex>

为页面添加页脚

```json
"plugins": [
   "page-footer-ex"
],
"pluginsConfig": {
    "page-footer-ex": {
        "copyright": "&copy 版权所有",
        "markdown": false,
        "update_label": "<b>更新时间：</b>",
        "update_format": "YYYY-MM-DD HH:mm:ss"
    }
}
```
